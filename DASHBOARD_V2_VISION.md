# 📊 SPÉCIFICATIONS : DASHBOARD "LE COCKPIT" V2.0 (VISION CEO)

Ce document détaille la vision ultime pour la refonte du Dashboard Analytics de BloLab CRM. L'objectif est clair : **Ce dashboard doit être la première chose que le Boss ou l'équipe a envie de regarder chaque matin en buvant son café.** Il doit offrir une vue d'ensemble chirurgicale de l'entreprise tout en permettant d'agir directement.

## 1. 🌅 MÉTÉO BUSINESS IA (L'Accueil du Boss)
*   **Objectif** : Un briefing exécutif proactif.
*   **Fonctionnement** : Un bloc premium en haut de page généré par l'IA.
*   **Composants** :
    - Humeur du business (ex: "Croissance forte", "Journée calme").
    - Résumé des nouveaux leads et alertes (ex: "5 nouvelles inscriptions cette nuit").
    - Suggestions des 3 actions prioritaires.

## 2. 🏢 VUE D'ENSEMBLE DE L'ENTREPRISE (La Tour de Contrôle)
L'interface doit respirer le professionnalisme. En un coup d'œil, on comprend la santé de BloLab.
*   **KPIs Actionnables (Drill-down)** :
    - Les cartes de statistiques deviennent des boutons interactifs.
    - Cliquer sur "Attente Humain" ouvre immédiatement un volet latéral avec les contacts bloqués.
*   **Tunnel de Conversion Interactif (Funnel)** : 
    - Visualisation graphique (Nouveau -> Qualifié -> Proposition -> Inscrit). 
    - Cliquer sur l'entonnoir filtre toutes les autres vues du dashboard.
*   **Radar des Objections** : Graphique d'extraction IA montrant ce qui bloque les ventes (Prix, Temps...).
*   **Jauge d'Objectifs (Nouveau !)** : Une barre de progression visuelle premium indiquant l'avancement vers les objectifs mensuels/annuels de l'entreprise.

## 3. 🎓 GESTION DYNAMIQUE DES PROGRAMMES (Les Hubs)
Le Dashboard n'est pas figé : il évolue avec l'offre de BloLab. Il permet de **Créer, Modifier et Supprimer des programmes entiers** directement depuis l'interface.
*   **Création de Programmes** : Le CEO peut ajouter un nouveau programme (ex: *Empow'her*, *ClassTech*, *Ecole229* avec ses sous-filières, *Incubateur*...) en un clic.
*   **Génération Automatique** : La création d'un programme génère automatiquement le "Hub" associé sur le dashboard.
*   **Vue par Hub (Mini-CRM)** : 
    - Chaque programme possède son onglet avec ses statistiques (Inscrits, Taux de conversion).
    - **Data Grid Interactif** : Un tableau listant tous les inscrits de CE programme spécifique.
    - **CRUD des Inscriptions** : Ajouter (⊕), Modifier (✎), et Supprimer (🗑️) des inscrits pour un programme donné directement depuis le dashboard. L'IA alimentera automatiquement ces listes lors de ses discussions.

## 4. ✍️ MODE ÉDITION GLOBALE (Manual Override)
*   **Objectif** : Flexibilité totale.
*   **Fonctionnement** : Le CRM ne doit jamais être bloquant. Les chiffres affichés peuvent être cliqués et modifiés manuellement pour refléter les ventes "hors-ligne" ou corriger le système.

## 🤖 5. LES "TRUCS CLASSES" SUPPLÉMENTAIRES (Effet Wahou)
*   **Export "One-Click PDF/Excel"** : Un bouton élégant `Générer le rapport` qui crée un PDF du briefing matinal et des chiffres clés, idéal pour les réunions de direction.
*   **Sidekick IA (Assistant Flottant)** : Une petite bulle qui "veille" sur le business et suggère des actions (ex: "Relancer Marc ?").
*   **Confettis / Sound Design** : Animations subtiles (micro-interactions) à chaque nouvelle inscription ajoutée ou objectif mensuel atteint.
